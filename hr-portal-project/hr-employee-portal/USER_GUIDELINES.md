# User Guidelines

This guide explains how HR users and Employees should use the portal.

## Start the Application

1. Ensure backend API is running.
2. Start the frontend app.
3. Open the application URL in your browser.

## Login

- Enter Email, Password, and User Type.
- Select the correct role:
  - HR
  - Employee
- Click Sign In.

If details are invalid, an error message is shown.

## HR User Guide

After HR login, a welcome screen is displayed with quick-access cards before any section is opened.

### HR Welcome Screen

- Shown automatically on first access to the HR dashboard.
- Displays your login email.
- Click any card to open that section directly:
  - Add Employee
  - View Employees
  - Leave Info
- The welcome screen reappears if you navigate back to the dashboard root.

### Add Employee

- Navigate to Add Employee.
- Enter employee email and department.
- Submit the form.
- You will see success or duplicate employee feedback.

### View Employees

- Navigate to View Employees.
- Review the list of employee email and department information.

### Leave Info

- Navigate to Leave Info.
- Review pending leave requests.
- Approve or Reject each request.

### Logout

- Use Logout button in dashboard header.

## Employee User Guide

After Employee login, a personalized welcome screen is displayed with quick-access cards before any section is opened.

### Employee Welcome Screen

- Shown automatically on first access to the Employee dashboard.
- Displays your first name and login email.
- Click any card to open that section directly:
  - Profile
  - Apply Leave
  - Leave Status
- The welcome screen reappears if you navigate back to the dashboard root.

### Profile

- Navigate to Profile.
- View your personal information: first name, last name, email, age, and department.

### Apply Leave

- Navigate to Apply Leave.
- Enter reason and number of leave days.
- Submit the request.

### Leave Status

- Navigate to Leave Status.
- Track status of each leave request:
  - Pending
  - Approved
  - Denied

### Logout

- Use Logout button in dashboard header.

## New Employee Sign Up

If you are not yet signed up:

1. Click Sign Up from login page.
2. Enter your registered company email.
3. If verified, complete personal details and password.
4. Submit to create your account.

## Best Practices

- Always select the correct user type at login.
- HR users should only approve/reject after reviewing leave reason and duration.
- Employees should keep password secure.
- Logout after finishing your session, especially on shared systems.
